# Docs Layout

## Canonical (kept in `Docs/` root)
- `ARCHITECTURE.md`
- `ERRORS.md`
- `ROADMAP.md`
- `NOTIFIERS_SIGNOFF.md`
- `PROJECT_MEMORIES.md`

## Sprints
- Sprint artifacts live under: `Docs/Sprints/Sprint-<N>/`

## PR drafts
- PR descriptions and drafts live under: `Docs/PRs/`

## Notifier patch notes
- Non-canonical notifier notes live under: `Docs/Notifiers/`

## Local-only notes (never committed)
- Local-only docs live under: `Docs/_local/` (ignored by git)
